# Animated Playground

A creative animated website built with HTML, CSS, and JavaScript.
This project focuses on experimentation, animations, and playful UI interactions.

## Features
- Smooth CSS animations
- Interactive UI elements
- Random visual effects
- Clean and minimal design
- No frameworks used

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## How to Run
1. Download or clone the repository
2. Open index.html in your browser
3. Explore and interact with the animations

## Purpose
This project was created to practice front-end development and
experiment with creative coding ideas.
